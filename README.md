# Building

To build the userscripts, you will need [NodeJS](https://nodejs.org/en/download/)

If you are on windows, download and unzip the repository and run `build.bat`

```sh
git clone https://github.com/e9x/kru.git

cd kru

npm install

node index.js -once
```
